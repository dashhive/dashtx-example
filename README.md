# Maya DASH Demo

1. Install [node.js](https://webinstall.dev/node)
   ```sh
   curl https://webi.sh/node | sh
   ```
2. Clone the repo:
   ```sh
   git clone git@gist.github.com:43a2a9d1007b6df2dd63eb9d1a5c1733.git
   pushd ./43a2a9d1007b6df2dd63eb9d1a5c1733/
   ```
3. Test the code:
   ```sh
   npm ci
   node ./maya-dash-demo.js
   ```
